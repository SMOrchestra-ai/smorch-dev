# English GTM — Folder Guide
## What this folder does
Produces 3 English LinkedIn posts about GTM, signal-based selling, Dream 100, and 7-11-4 framework from any transcript, with visuals, pushed to GHL as drafts.
## Files in this folder
- mamoun-founder.md — who Mamoun is, his rules, his voice
- brand-start.md — the trigger and full workflow
- instructions.md — detailed step-by-step process
- skill-english-gtm.md — quality standards and requirements
- good-lookalikes/ — PDF examples to model every post against
- templates/ — approved posts saved here after review
## How to use
1. Open this folder in Cowork
2. Say: "Brand Start"
3. Wait for: "English GTM loaded. Ready."
4. Paste a transcript (VTT, SRT, plain text, or PDF)
5. Everything else happens automatically
## Output
- 3 English GTM posts
- 3 visuals (Gemini or Canva based on content)
- All saved in posts-[DATE] folder
- All saved as drafts in GHL under Mamoun Alamouri personal LinkedIn
- Never published directly — always review first
## Rules that never change
- No em dashes
- No hype, no guru content, no gimmicks
- Specific numbers and real outcomes only
- MENA context always
- Match good-lookalikes quality or rewrite
- Carousel via Canva, infographic and single visual via Gemini
## After approval
When a post is approved say:
"This post is approved. Save it as template to English-GTM/templates/template-approved.md"
