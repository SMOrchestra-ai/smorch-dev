# Skill — English AI Claude Content Creator

## What this skill does
Transforms a YouTube video into 3 high-quality English LinkedIn posts about AI and Claude, with visuals, pushed to GHL as drafts.

## When to use this skill
- User says "Start" or "Brand Start" and provides a transcript
- User shares a transcript (VTT, SRT, plain text, or PDF)
- User asks to create LinkedIn posts from a transcript

## Inputs required
- Transcript (VTT, SRT, plain text, or PDF) — never a YouTube link or video
- Access to good-lookalikes/ folder
- Access to mamoun-founder.md

## Quality standard
Every post is measured against 2 benchmarks:
1. mamoun-founder.md rules — non-negotiable
2. good-lookalikes PDFs — style, length, structure, depth

3. Quality gate score — minimum 9/10 average before shipping

If a post does not pass all 3 — rewrite before delivering. Never deliver below 9/10.

## Post requirements
- Topic: AI tools, Claude, automation in real business
- Length: 150-3000 characters
- Structure: hook + context bridge + numbered breakdown + closing punch + DM CTA + P.S.
- Voice: first person, founder-to-founder, contrarian, opinionated
- Numbers: always specific — never vague
- MENA angle: include when relevant
- MicroSaaS angle: include where relevant
- No em dashes, no hashtags, no emojis, no fluff, no hype, no guru content

## Visual requirements
For each post:
- Study good-lookalikes PDFs for design style first
- Generate a Gemini image prompt that matches:
  - The post topic and hook
  - The exact style from good-lookalikes (colors, layout, typography)
  - Clean, professional, scroll-stopping
  - No stock photo feel
- Send prompt to Gemini and attach the generated image to the post

## Output
- 3 English posts
- 3 visuals (one per post)
- All pushed to GHL as drafts
- Never published directly

## Anti-gimmick check (mandatory before delivery)
Ask for every post:
- Does it sound like Mamoun wrote it?
- Is it specific — real numbers, real tools, real outcomes?
- Does it match good-lookalikes quality and length?
- Could anyone else have written this? If yes — rewrite.
- Does it sound like hype or guru content? If yes — kill it.
