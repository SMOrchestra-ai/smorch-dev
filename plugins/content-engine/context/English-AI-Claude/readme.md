# English AI Claude — Folder Guide

## What this folder does
Produces 3 English LinkedIn posts about AI and Claude from any transcript, with visuals, pushed to GHL as drafts.

## Files in this folder
- mamoun-founder.md — who Mamoun is, his rules, his voice
- brand-start.md — the trigger and full workflow
- instructions.md — detailed step-by-step process
- skill-english-ai-claude.md — quality standards and requirements
- good-lookalikes/ — PDF examples to model every post against

## How to use
1. Open this folder in Cowork
2. Say: "Brand Start"
3. Wait for: "English AI Claude loaded. Ready."
4. Paste a transcript (VTT, SRT, plain text, or PDF)
5. Everything else happens automatically

## Output
- 3 English posts about AI and Claude
- 3 visuals generated by Gemini
- All saved as drafts in GHL Social Planner
- Never published directly — always review first

## Rules that never change
- No em dashes
- No hype, no guru content, no gimmicks
- Specific numbers and real outcomes only
- MENA context always
- Match good-lookalikes quality or rewrite
