# Visual Spec — Post 1: Claude as Scoring Partner

**Format:** Carousel (8 slides)
**Dimensions:** 1080x1080 (1:1)
**Tool:** HTML carousel provided (carousel.html) — user recreates in Canva

## Design System
- Background: solid black (#000000)
- Accent: orange (#FF6600)
- Text: white (#FFFFFF)
- Font: Cairo (Google Fonts, Arabic RTL)
- MA logo: orange circle with "MA" in white, top-left corner, every slide
- Pagination dots: bottom center, orange active dot
- Subtle grid overlay: 40px spacing, rgba(255,102,0,0.03)
- Orange glow effects on key slides (radial gradient)
