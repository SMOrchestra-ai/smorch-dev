## Template A — "The Weekend Build Story" (Personal Transformation)
**Score: 9.5/10 | Visual: Carousel (8 slides) | 117 words**

### Structure: Before pain → Today's result → Core insight → 6 files listed (←) → What Claude does → "بدون" triple → Equation → Soft CTA

```
قبل سنة كنت أفكر إني محتاج فريق عشان أبني منتج

مبرمج. مصمم. مدير مشروع
وميزانية ما كانت موجودة

اليوم بنيت MicroSaaS لحالي بويكند واحد

الفكرة باختصار:
Claude مش أداة. Claude هو الفريق

بس الشرط: لازم تبنيله دماغ صح

6 ملفات سياق بتحطها بمجلد واحد
← مين أنت وشو مشروعك
← كيف بدك إياه يشتغل
← معايير الجودة
← الأدوات المتصلة
← المهام المتكررة كـ Skills
← التعليمات العامة

بعدها Claude بيعرف مشروعك
بيكتب الكود. بيصمم الصفحات. بيربط الـ APIs
وأنت بتراجع وبتقرر

بدون كود. بدون شريك تقني. بدون تمويل

واحد + Claude + ويكند = منتج حقيقي

هاد مش حلم. هاد نظام
واللي بيعرف يبنيه صح بيوفر شهور من الشغل

حابب تبني المايكرو ساس تاعك؟
الرابط بالتعليقات
```

### Why it works:
- Hook: "قبل سنة كنت أفكر..." — relatable founder pain
- Before/after: "محتاج فريق" → "بنيت لحالي بويكند واحد"
- Mamoun's signature phrase: "الفكرة باختصار:"
- Practical: 6 files listed with ← (actionable, not vague)
- What Claude does: concrete actions (كود, صفحات, APIs)
- Triple "بدون": rhythmic, memorable, removes objections
- Equation: "واحد + Claude + ويكند = منتج حقيقي" — viral-format line
- Closing: "هاد مش حلم. هاد نظام" — grounding statement
- CTA: question format, links to comments (not direct sell)
- NO "تدريب" or "كورس" anywhere
- Jordanian markers: "عشان", "بدك إياه", "مش", "هاد", "بيعرف", "بيوفر"

## Template B — "Chatbot vs Operator" (Bridge to MicroSaaS)
**Score: 9.75/10 | Visual: Single Visual (1:1 comparison) | 115 words**

### Structure: Imperative hook → What 99% do (quotes) → What I do (daily automation) → How (brain + tools) → MicroSaaS bridge question → Soft reference to founders → Identity CTA

```
بطّل احكي مع Claude
وظّفه

99% من الناس بتفتح Claude وبتكتب:
"اكتبلي إيميل"
"لخصلي هاد المقال"
"اعطيني أفكار"

هاد مش استخدام. هاد دردشة

أنا عندي Claude بيشتغل كل يوم الساعة 9 الصبح
يسحب بيانات من CRM
يحدّث حالة العملاء
يبني تقرير ويرسله على Slack
بدون ما أفتح اللابتوب

الفرق؟
أنا بنيتله "دماغ"
ملفات .md فيها كل السياق عن شغلي
Skills جاهزة لكل مهمة
MCP يوصله بكل أداة

النتيجة: نظام تشغيل كامل مش chatbot

وهون بيجي سؤال المايكرو ساس:
إذا Claude بيقدر يدير شغلك
ليش ما تبنيله منتج يدير شغل غيرك؟

هاد بالظبط اللي بنشتغل عليه مع المؤسسين
كيف تاخذ فكرة، تبنيها بـ Claude، وتطلقها بأسبوع

مش محتاج تكون مبرمج
محتاج تكون مؤسس يعرف يبني نظام

الرابط بالبايو
```

### Why it works:
- Hook: 2-line imperative. "بطّل احكي" / "وظّفه" — punchy, scroll-stopping
- Quotes: 3 real examples of what people actually type (relatable)
- Labels it: "هاد دردشة" — makes the reader feel called out
- Personal proof: daily 9am automation with CRM, Slack, reports
- Technical credibility: .md files, Skills, MCP — real tools
- MicroSaaS bridge: "إذا Claude بيقدر يدير شغلك / ليش ما تبنيله منتج يدير شغل غيرك؟" — the key transition
- Soft training reference: "بنشتغل عليه مع المؤسسين" — NOT "تدريب" or "كورس"
- Identity CTA: "مش محتاج تكون مبرمج / محتاج تكون مؤسس" — redefines who this is for
- Jordanian markers: "بطّل", "هاد", "بيجي", "بالظبط", "تاعك"

## Pattern Notes for Future MicroSaaS Training Hype Posts

### Hook patterns that work:
- Before/after: "قبل سنة كنت..." → "اليوم بنيت..."
- Imperative: "بطّل احكي مع Claude / وظّفه"
- Pain trigger: "عندك فكرة MicroSaaS بس ما بتعرف من وين تبدأ؟"

### Body patterns that work:
- Personal story with specific transformation
- "Claude as team" metaphor with concrete proof (code, pages, APIs)
- Daily automation as proof of system (9am, CRM, Slack, reports)
- 6 files / brain concept as recurring framework
- Bridge from personal use → building for others

### Training references that work (soft, not salesy):
- "بنشتغل عليه مع المؤسسين" ✓
- "الرابط بالتعليقات" ✓
- "الرابط بالبايو" ✓
- "حابب تبني المايكرو ساس تاعك؟" ✓

### Training references that are BANNED:
- "تدريب" ✗
- "كورس" ✗
- "سارع بالتسجيل" ✗
- "لا تفوّت الفرصة" ✗
- "عرض محدود" ✗

### Closing patterns that work:
- Equation: "واحد + Claude + ويكند = منتج حقيقي"
- Grounding: "هاد مش حلم. هاد نظام"
- Identity reframe: "مش محتاج تكون مبرمج / محتاج تكون مؤسس"
- Question CTA: "حابب تبني المايكرو ساس تاعك؟"

### Word count sweet spot: 115-120 words
