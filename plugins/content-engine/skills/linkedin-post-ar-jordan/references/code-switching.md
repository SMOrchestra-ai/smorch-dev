# Code-Switching — متى تستخدم الإنجليزي

## دايماً بالإنجليزي
- الأدوات: Claude, Cowork, n8n, GHL, Clay, Claude Code
- المصطلحات: Context, SOP, ICP, MicroSaaS, GTM, MVP, API
- المنصات: LinkedIn, WhatsApp, YouTube

## دايماً بالعربي الأردني
- الأفعال: "بنيت" مش "built"
- الوصف: "طلع معي" مش "I got"
- المشاعر والسياق

## مثال صح
"عملت context file لـ Claude وطلع معي pipeline أحسن بـ ٣ مرات."

## مثال غلط
"I built a context file and the pipeline improved" ← إنجليزي كامل
"قمتُ ببناء ملف السياق وتحسّن خط الأنابيب" ← فصحى رسمية
