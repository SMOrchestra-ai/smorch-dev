# Jordanian Arabic Markers

## Core Replacements
| الفصحى | الأردني |
|---|---|
| هذا | هاد |
| هذه | هاي |
| هؤلاء | هدول |
| الآن | هلأ |
| أريد | بدي |
| يجب | لازم |
| سوف | رح |
| ماذا | شو |
| أين | وين |
| لماذا | ليش |
| استمر | ضل |
| جيد | منيح |
| فقط | بس |
| أيضاً | كمان |
| لذلك | عشان هيك / فـ |
| في النهاية | بالآخر |
| بالضبط | بالظبط |

## عبارات مأمون الخاصة
- "الفكرة باختصار:"
- "هاد هو السر:"
- "مش بالـ [X]. مش بالـ [Y]."
- "احفظه. طبّق اليوم."
- "ارجعلي بعد أسبوع احكيلي شو تغيّر."
- "نبني ونطلق مع بعض."
- "هذا هو الـ [X]."
- "هاد مش استشارة. هاد نظام."

## اختبار اللهجة
اقرأ البوست بصوت عالٍ.
هل يبدو طبيعي لو حدا أردني قاله في محادثة عادية؟
هل فيه أي كلمة تشبه خبر إذاعي؟
إذا آه → أعد الكتابة.
