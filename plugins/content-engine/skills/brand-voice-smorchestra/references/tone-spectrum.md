# Tone Spectrum — Mamoun Alamouri

## How to Choose

| Reader state | Best tone |
|---|---|
| Doing something wrong, doesn't know it | Contrarian |
| Looking for a better system | Authority |
| Feeling stuck or alone | Narrative |

Never mix tones in one post.

---

## Contrarian
Use when: topic has a widely-held belief that's wrong.
Structure: State broken belief → why it fails → show better way → challenge
Example hook: "Most MENA GTM fails before the first meeting."
Example hook: "Cold outreach is not dead. Bad cold outreach is dead."

## Authority
Use when: sharing a system, framework, or result.
Structure: State result → numbered breakdown → proof → DM CTA
Example hook: "I replaced 3 SDRs with one signal workflow. Here's the exact stack."
Example hook: "We closed $180K in MENA pipeline without a single cold call."

## Narrative
Use when: sharing a personal lesson or turning point.
Structure: Before state → moment of change → after state → lesson → question
Example hook: "A year ago I had 47 coffee meetings and zero pipeline."
Example hook: "I almost shut down SMOrchestra in Q1. Here's what saved it."
